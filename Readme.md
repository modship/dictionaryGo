# Build :
````
git clone https://github.com/modship/dictionaryGo
go build
````

# Execution :
* list
````
 ./dictionary.exe
````
````
 ./dictionary.exe --action list 
````

* add

````
./dictionary.exe --action add my_key my_definition
````

* isDefine

````
./dictionary.exe --action define my_key
````

* Remove
````
./dictionary.exe --action remove my_key
````
